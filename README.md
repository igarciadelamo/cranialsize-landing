# CranialSize — Landing Page

Marketing and informational landing page for [CranialSize](https://github.com/igarciadelamo/cranialsize), a web application for osteopaths to track and analyze infant cranial measurements.

## About CranialSize

CranialSize helps pediatric osteopaths:
- Track cranial circumference measurements over time
- Visualize growth against WHO percentile curves (P3–P97), sex-specific
- Manage patient records securely with Google authentication

## Tech Stack

Single HTML file with embedded CSS and vanilla JS. No build step, no dependencies.

## Getting Started

1. Clone the repository
```bash
git clone git@cranialsize-github:igarciadelamo/cranialsize-landing.git
cd cranialsize-landing
```

2. Open `index.html` in a browser — that's it.
